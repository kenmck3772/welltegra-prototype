Data Schema

Place a JSON array at /data/wells.json. Each row represents a well-month.

[{
  "well_name": "ST12-03A",
  "field": "Stromness",
  "operator": "WellTegra",
  "month": "2024-05-01",
  "oil_bbl": 18765,
  "water_bbl": 6240,
  "gas_mcf": 312,
  "lat": 57.095, "lon": -2.241
}]

Optional fields are acceptable; missing values are treated as 0. Add lat/lon to enable map cards in future.
